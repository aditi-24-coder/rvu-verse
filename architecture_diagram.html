Testing
Test File Description
test_user.py Validates that user registration stores the correct email domain.
test_Profile.py Verifies that a Profile is automatically created when a User is saved.
test_post.py Confirms that a post with an image is saved correctly.
test_comment.py Verifies that comments are linked to the correct post.
test_like_count.py Checks that the like counter increments correctly.
test_follow.py Tests the follow and unfollow flow.
test_hashtags.py Validates the extract_hashtags() method on Post.
test_notification.py Confirms notifications are created on relevant events.
test_event_registration.py Verifies the default registration status.
test_models.py Tests Department model string representations.
test_message.py Validates that messages are created and default to unread status.
test_basic.py Basic arithmetic sanity check confirming pytest is working.
test_integration.py Integration suite simulating full user flows (login, post creation, and mutual-follow messaging).
